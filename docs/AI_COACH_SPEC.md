# AI 재무 코치 — thin slice 스펙 (v2)

> 가드레일 우선. 구현 전 이 문서를 계약으로 본다.

## 목적
정적 시뮬레이터를 **배분 구조 코치**로 승격. Freemium 웨지.

## In scope (v2 thin)
- 입력: `FinancialSnapshot` + `buckets` + `projectEngine` 요약(ETA·달성·crossover)
- 출력: 3~5문장 설명 + “왜?” 1단 근거 (배분·저축률·비상금 구조)
- UI: 엔진 결과 패널 / 홈 다음 한 걸음 옆 “코치에게 물어보기”

## Out of scope (금지)
- 개별 종목·매물·지역 추천
- 레버리지·대출 권유
- 수익 보장·확정 표현
- 목표 없이 억 단위 확신을 강화하는 카피

## 실패 모드 → 거부 템플릿
| 조건 | 응답 |
|---|---|
| snapshot/엔진 비어 있음 | 현황·배분을 먼저 넣어 달라고 안내 |
| 숫자 목표 없음 | 목표 참고선을 정하자고 안내 (곡선 해석 보류) |
| 모델이 종목/매물 언급 | 서버/클라이언트 필터로 차단 후 재생성 |
| 수익률 단정 | “예시·가정” 강제 접두 |
