# 시라노 — 화면 용어 (쉬운 말)

내부 코드명과 사용자에게 보이는 말을 분리한다. **UI에는 오른쪽만 쓴다.**

| 쓰지 말 것 (내부/옛) | 화면에 쓸 말 |
|---|---|
| 엔진 / 재테크 엔진 | **자산 설계** |
| 공방 | (메뉴 그룹명 제거) |
| 팔레트 | **항목 추가** |
| 버킷 | **항목** |
| 인스펙터 | **항목 수정** |
| 허브 / 루프 / 준비 | (그룹 라벨 제거, 메뉴만) |
| Upstream / Downstream | 수입 → 배분 → 자산 |
| 그룹 / 부모 노드 | **묶음** |
| 리프 버킷 | **세부 항목** (또는 그냥 **항목**) |
| ratioOfParent | **상위 대비 %** (루트면 **수입 대비 %**) |
| ratioOfTotal | **전체 대비 %** (자동 계산) |

코드·테스트·계산 로직의 `engine` / `bucket` 식별자는 유지해도 된다.
