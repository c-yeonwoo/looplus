# 자산 설계 — 계층 배분 UX

## 모델

```
수입 항목들 (근로·자본·자산 현금흐름…)  →  월수입
                                              ├─ 투자 / 저축  →  자산
                                              │                    ↖ 현금흐름(점선) → 수입원
                                              └─ 지출 (아래 밴드)
                                                   ├─ 고정
                                                   └─ 변동
```

- 수입 항목: **월 금액(만원)** 입력 → 월수입에 모임 → downstream 배분의 기준.
- 배분 항목: **상위 대비 %** 만 설정. 엔진이 전체 %·월 환산(버림) 표시.
- **자산** = 투자·저축 리프가 모이는 곳. 복리는 자산 안. 실현 현금흐름은 점선으로 수입원(또는 월수입)에 재유입.
- 자산 노드에서 **수입원으로 연결**하면 `자산 현금흐름` 자본소득 노드가 생김/갱신됨.
- 프로젝션은 배분 **리프**만 사용.

## 캔버스

- L→R: 수입 항목 → 월수입 → 투자/저축 → 자산. 지출은 아래 밴드.
- **휠 줌** (툴바 % 클릭 시 100% 복귀).
- **선택**: 클릭 단일 · Shift/⌘/Ctrl+클릭 추가 · 마퀴 복수 선택 · 일괄 드래그.
- **드래그**: 노드 위치 · 선 핸들은 곡선만. 「자동 정렬」원위치.
