# 자산 설계 — 계층 배분 UX

## 모델

```
월 수입
 ├─ 투자 (수입 대비 %)
 │   ├─ 주식 (투자 대비 %)
 │   └─ 연금 (투자 대비 %)
 ├─ 저축 …
 └─ 지출 …
     ├─ 고정지출
     └─ 변동지출
```

- 유저가 넣는 값은 **바로 위 노드 대비 %** 만.
- 엔진이 **전체 대비 %**(경로 곱)와 **월 n만**(수입 × 전체% / 100, **소수 버림**)을 표시.
- 프로젝션은 **리프(끝 항목)** 만 사용. 묶음은 비율 컨테이너.

## 캔버스 (L→R 트리)

- SVG 그래프: 수입(좌) → 묶음/세부 → 모인 자산(우).
- 하위 항목(depth≥2)은 더 작은 카드로 위계 표시.
- 부모–자식·유입 링크는 곡선 + 흐름 애니메이션.
- **드래그**: 항목 노드를 끌어 위치 조절 (`canvasX`/`canvasY`). 「자동 정렬」로 원위치.
- **마인드맵식 포커스**: 클릭 선택 ·「하위」로 가지 추가.
- 삭제는 서비스 내 확인 모달.

## 팔레트 ↔ 캔버스

| 동작 | 결과 |
|---|---|
| 묶음 클릭 / 수입의「묶음」 | 수입 바로 아래 루트 |
| 세부 항목 · 하위 추가 | **포커스 노드 아래** |
| 캔버스 × / 항목 수정 삭제 | 확인 모달 후 삭제(하위 포함) |
| 추천 배분 | 위 트리 초안 |

포커스 = “다음에 어디에 붙일지”. 노드를 옮겨 다니며 가지를 뻗는 흐름.
