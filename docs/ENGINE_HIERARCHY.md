# 자산 설계 — 계층 배분 UX

## 모델

```
월 수입
 ├─ 투자 (수입 대비 %)
 │   ├─ 주식 (투자 대비 %)
 │   └─ 연금 (투자 대비 %)
 ├─ 저축 …
 └─ 지출 …
     ├─ 고정지출
     └─ 변동지출
```

- 유저가 넣는 값은 **바로 위 노드 대비 %** 만.
- 엔진이 **전체 대비 %**(경로 곱)와 **월 n만**(수입 × 전체% / 100, **소수 버림**)을 표시.
- 프로젝션은 **리프(끝 항목)** 만 사용. 묶음은 비율 컨테이너.

## 팔레트 ↔ 캔버스

| 동작 | 결과 |
|---|---|
| 묶음(투자/저축/지출) 클릭 | 수입 바로 아래 루트 추가 |
| 세부 항목 클릭 | **선택 중인 항목 아래**에 추가. 선택 없으면 같은 카테고리 루트 아래. 그것도 없으면 루트 |
| 캔버스 × | 확인창 후 삭제(하위 포함) |
| 추천 배분 | 위 트리 초안 생성 |

선택 = “다음에 어디에 붙일지” 컨텍스트. 묶음을 먼저 고른 뒤 세부 항목을 누르는 흐름이 자연스럽다.
