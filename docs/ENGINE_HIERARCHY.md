# 자산 설계 — 계층 배분 UX

## 모델

```
수입 항목들 (근로·자본·…)  →  월수입
                                ├─ 성장 / 안전  →  자산(복리)
                                │                    ↖ 현금흐름(점선) → 월수입
                                └─ 지출 (아래 밴드)
                                     ├─ 고정
                                     └─ 변동
```

- 수입 항목: **월 금액(만원)** 입력 → 월수입에 모임 → downstream 배분의 기준.
- 배분 항목: **상위 대비 %** 만 설정. 엔진이 전체 %·월 환산(버림) 표시.
- **자산** = 성장·안전 리프가 모이는 곳. **복리는 자산 안에서만.** 배당·임대 등 실현 현금흐름만 점선으로 월수입에 재유입.
- 프로젝션은 배분 **리프**만 사용.

## 캔버스

- L→R: 수입 항목 → 월수입 → 성장/안전 → 자산. 지출은 아래 밴드.
- 하위(depth≥2)는 작은 카드.
- **선택**: 클릭 단일 · Shift/⌘/Ctrl+클릭 추가 · **빈 곳 드래그=마퀴(박스) 복수 선택** · 복수 선택 시 일괄 드래그.
- **드래그**: 수입 항목·월수입·배분·자산 노드 위치 조절. **선·가운데 핸들**은 곡선만 휘게 함(노드 고정). 「자동 정렬」은 노드·선 제어점 모두 초기화.
- 월수입: 기본 간소화, **클릭 시**「묶음」·우측 패널에서 수입 항목 추가.
- 같은 레이어 **위로/아래로**로 순서 변경 (형제 `position`).

## 팔레트

| 동작 | 결과 |
|---|---|
| **수입원** | 월수입 왼쪽. 프리셋·직접 만들기 모두 **금액 입력 후** 추가 |
| 묶음 | 월수입 바로 아래 루트 (성장·안전·지출) |
| 세부 / 하위 | 포커스 노드 아래 |
| 삭제 | 확인 모달 (하위 포함) |
