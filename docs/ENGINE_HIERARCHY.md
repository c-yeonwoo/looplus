# 자산 설계 — 계층 배분 UX

## 모델

```
수입 항목들 (근로·자본·…)  →  월 수입(합산)
                                ├─ 투자 / 저축  →  투자·저축 합류(중간 집계)
                                └─ 지출 (아래 밴드)
                                     ├─ 고정
                                     └─ 변동
```

- 수입 항목: **월 금액(만원)** 입력 → 월 수입에 합산 → downstream 배분의 기준.
- 배분 항목: **상위 대비 %** 만 설정. 엔진이 전체 %·월 환산(버림) 표시.
- **모인 자산 / 투자·저축 합류** = 최종 싱크가 아니라 중간 집계. 실현 수익은 점선으로 월 수입 재유입.
- 프로젝션은 배분 **리프**만 사용.

## 캔버스

- L→R: 수입 항목 → 월 수입 → 배분 → 합류.
- 지출 루트는 **아래 밴드**에 자동 배치 (성장 밴드와 분리).
- 하위(depth≥2)는 작은 카드.
- **드래그**: 수입 항목·월 수입·배분·합류 모두 위치 조절. 「자동 정렬」원위치.
- 월 수입: 기본 간소화, **클릭 시**「묶음」·우측 패널에서 수입 항목 추가.
- 같은 레이어 **위로/아래로**로 순서 변경 (형제 `position`).

## 팔레트

| 동작 | 결과 |
|---|---|
| **수입원** | 월수입 왼쪽. 프리셋·직접 만들기 모두 **금액 입력 후** 추가 |
| 묶음 | 월 수입 바로 아래 루트 |
| 세부 / 하위 | 포커스 노드 아래 |
| 삭제 | 확인 모달 (하위 포함) |
